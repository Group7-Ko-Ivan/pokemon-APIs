# Pokemon

<br>

## Show Pokemon Trading Card

---

Returns All Pokemon Trading Card List

* URL: `/card`

* Method: `GET`

* Request Params: `None`

* Request Body: `None`

* Success Response:

  * **Code**: `200`
  * **Content**:
    >`{"output": [
        "https://images.pokemontcg.io/pl1/1.png",
        "https://images.pokemontcg.io/det1/1.png"
        }`

<br>

* **Error Response**
  * **Code**: `500`
  * **Content**: `Internal Server Error`
